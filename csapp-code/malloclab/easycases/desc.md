# 一、MALLOC 核心测试用例（4 个部分）

1. **首次分配** → 测试堆初始化 + 自动扩容
2. **常规分配** → 测试找到空闲块、正常切割
3. **大块分配** → 测试多次扩容、容量适配
4. **无合适块** → 测试自动扩容新块

# 二、FREE 核心测试用例（4 个部分）

1. **前后都已分配** → 直接插入空闲链表
2. **仅前面空闲** → 向前合并，仅更新大小
3. **仅后面空闲** → 向后合并，更新链表指针
4. **前后都空闲** → 三块合并，全局更新链表
