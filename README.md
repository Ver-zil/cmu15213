# CMU 15-213 CSAPP
## 项目简介
- 课程：CMU 15-213 / CSAPP 深入理解计算机系统
- 编程语言：C、x86-64汇编
- 所属大学：卡内基梅隆大学

## 本人工作
完成 7 个核心 Lab
- [x] DataLab：位运算与整数/浮点数逻辑
- [x] BombLab：反汇编与调试
- [x] AttackLab：栈溢出与代码注入
- [ ] ArchLab：Y86 汇编与 CPU 处理器设计
- [x] CacheLab：缓存模拟与优化
- [x] ShellLab：简易 Shell 实现
- [x] MallocLab：动态内存分配器
- [x] ProxyLab：并发缓存 Web 代理

## 环境依赖
- OS：Linux(x86_64) / WSL2(Ubuntu)
- GCC：9.3.0 ~ 9.4.0
- Make：≥3.8
